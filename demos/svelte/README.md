# Svelte + TypeScript + Vite

Minimal UseClassy demo for Svelte. Shows:

- Quoted UseClassy modifiers (`class:hover="..."`, `class:md="..."`)
- Native Svelte class directives (`class:scale-110={pressed}`) left untouched

```bash
pnpm --filter svelte-demo dev
```

Runs on [http://localhost:3002](http://localhost:3002).
